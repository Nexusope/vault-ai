# UI Specification

Control Room prioritizes the daily briefing, four system metrics, opportunity radar, captures, and pipeline status. Library supports query, media filters, grid/list, selection, and empty/error/loading states. Galaxy provides 3D orbit/zoom plus Timeline, Kanban, Network, Heatmap, Focus controls and a keyboard-readable list. Fusion selects 2–5 ideas, a constraint, staged generation, and structured production output.

AI Operator shows context sources, provider route, citations, streaming state, suggestions, and retry behavior. Deep Search exposes hybrid/semantic/keyword modes and explains every match. Collections support creation, rename/delete confirmation, drag assignment, and counts. Intelligence shows velocity, knowledge mix, weekly insight, and system score. Notifications group trend, fusion, graph, import, and security events. Settings owns CLI status, provider preference, storage health, notifications, account, and theme.

Every route defines skeleton, no-data, filtered-empty, recoverable error, permission error, and offline state. Mobile collapses the sidebar into a drawer, makes Fusion sequential, replaces forced 3D with the list fallback, keeps capture available, and avoids horizontal content overflow. Keyboard shortcuts must not fire inside text fields; Escape closes the topmost surface; focus returns to its trigger.
