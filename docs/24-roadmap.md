# Roadmap

Phase 0 (complete in this repository): high-fidelity ten-screen frontend, D1 schema/migration, idea API, provider abstraction, local Instagram CLI bridge, responsive 3D Galaxy, and buildable Sites runtime.

Phase 1: user-authenticated capture, real Library CRUD, CLI connection onboarding, media upload to R2, import jobs, transcript/analysis pipeline, live assistant routing, and browser critical-path tests. Exit requires ten real imports, deletion/export, and provider fallback evidence.

Phase 2: Qdrant hybrid search, persisted graph, Fusion production packs, personal trend engine, notifications, and recommendation feedback. Exit requires measured search/recommendation quality and cost budgets.

Phase 3: PostgreSQL/NestJS/Redis/BullMQ enterprise services, organizations/RBAC, audit, billing, multi-region read paths, extension, and mobile companion. Phase 4 adds additional social/document sources only after policy and adapter tests. No roadmap item is labeled complete without runtime evidence and migration/runbook coverage.
