# Future Features

Source expansion includes TikTok, YouTube, Pinterest, Threads, X, Facebook, Reddit, websites, PDFs, screenshots, images, voice notes, and notes through the `SourceAdapter` boundary. Each adapter requires terms review, authorization model, normalized metadata mapping, rate-limit plan, deletion behavior, and contract tests.

Product expansion includes collaborative workspaces, browser extension, mobile capture/share sheet, publishing calendar, versioned scripts, team comments, enterprise SSO/SCIM, policy retention, audit export, regional residency, and customer-managed keys. Intelligence expansion includes multimodal embeddings, temporal graph queries, causal trend confidence, experiment tracking, and user-trainable recommendation controls.

Features enter implementation only with a user outcome, privacy classification, source-of-truth design, performance/cost budget, abuse analysis, migration, observability, rollback, and measurable exit criteria. “Coming soon” UI never implies an integration is configured.
