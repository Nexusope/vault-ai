# Lovable Build Prompt — Vault AI

Build the complete frontend for Vault AI, an AI-powered Creative Operating System for creators. Produce UI only: routing, design system, components, layouts, interactions, animations, accessibility, and realistic typed mock data. Do not invent authentication, database, billing, or server behavior. Real data boundaries must be typed placeholder fetch functions that can be replaced without restructuring components. Never include API keys.

Use Next.js App Router, React, TypeScript, Tailwind, shadcn/ui, Framer Motion, GSAP only for justified complex sequences, Three.js through React Three Fiber/Drei for the Galaxy, Zustand, and TanStack Query. Create a tokenized light/dark theme with color, typography, spacing, radius, elevation, and motion tokens; no hardcoded component colors. The visual direction is high-quality technical brutalism: black dominant surfaces, sharp grid lines, luminous signal green, cyan semantic data, orange urgent opportunity, oversized typography, and dense monospaced metadata. It must feel premium and intentional, not like a generic admin template.

Build navigable routes for Dashboard, Library, Idea Galaxy, Idea Fusion, AI Assistant, Search, Collections, Analytics, Settings, and Notifications. Every route needs desktop/tablet/mobile layouts plus loading, empty, filtered-empty, error, offline, and permission states. Dashboard includes daily AI briefing, opportunity radar, recent captures, quick actions, and pipeline status. Library includes grid/list, query, filters, sort, selection, and bulk actions. Search supports keyword/semantic/hybrid modes with explicit match reasons.

Idea Galaxy must use R3F for desktop 3D nodes, clusters, particles, orbit/zoom/pan, filters, search, and Galaxy/Timeline/Kanban/Library/Network/Heatmap/Opportunity/Focus modes. Provide an equally capable keyboard-readable 2D/list fallback and default to it on mobile/low-power/reduced-motion. Fusion is a drag/select canvas for 2–5 idea cards and a staged mocked result with hook, script, storyboard, shot list, thumbnails, captions, hashtags, audio, schedule, and alternatives. Mobile Fusion becomes a step flow.

AI Assistant simulates streaming messages, suggested prompts, cited saved ideas, context-source controls, provider route status, cancel/retry, and error recovery. Settings shows Instagram CLI as a local connection with explicit login-required status; other sources are coming soon. Show OpenRouter, NVIDIA, Groq, Gemini, and Cerebras as routing preferences, never key inputs.

Create reusable IdeaCard, IdeaNode, GalaxyCanvas, FusionPanel, TrendBadge, ChatBubble/Composer, SearchBar, Sidebar, Timeline, CollectionCard, MediaViewer, TranscriptPanel, InsightCard, EmptyState, ErrorState, and list/grid skeletons with typed props and variants. Use semantic HTML, complete keyboard navigation, visible focus, ARIA for custom/3D interactions, WCAG AA contrast, 44px mobile targets, and `prefers-reduced-motion`. Deliver a polished runnable frontend; do not leave placeholder text, broken controls, starter metadata, or console errors.
