# Independent QA Prompt

Test the running Vault AI application independently against the product/UI/API requirements. Do not trust the implementer’s completion claim. Check every route at desktop and mobile, keyboard navigation, visible focus, reduced motion, dialog focus/close, Library query/grid/list/selection/empty, Galaxy rendering and fallback, two-source Fusion through result, Assistant success/fallback, Search modes/results, Collections, Analytics, Notifications, provider toggles, CLI bridge state, capture persistence, API validation, health, console errors, broken links, overflow, and accessibility names. Report each finding with severity, route, exact steps, expected, actual, and evidence. Re-test fixes and clearly separate verified, contradicted, and unverified requirements.
