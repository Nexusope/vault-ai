# Vault AI Agent Entry Point

All agents must read `agents/main-orchestrator.md` before changing this repository, then the role file matching their assignment and the directly relevant numbered docs. The orchestrator defines ownership, sequencing, integration, testing, failure handling, and completion. Never expose secrets, perform Instagram login for the user, invent undocumented CLI capabilities, or call partial setup production-ready. A task is done only when its requirement-to-evidence audit and relevant runtime checks pass.
